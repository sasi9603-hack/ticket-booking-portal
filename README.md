# TicketBookEra – Multi-Service Booking Platform

TicketBookEra is a front-end web application that provides a unified interface for booking multiple services including trains, buses, hotels, movie tickets, and food orders.

## Features
- Train Ticket Booking Interface
- Bus Ticket Booking Interface
- Hotel Reservation Form
- Movie Ticket Booking Section
- Food Ordering Interface
- Dynamic content switching using JavaScript
- Responsive UI with Bootstrap

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap

## Project Structure
Home.html – Main webpage layout
Home.css – Custom styling
MakeTKTjscript.js – JavaScript for dynamic section switching

## Future Improvements
- Backend integration
- Real-time booking APIs
- Payment gateway integration
